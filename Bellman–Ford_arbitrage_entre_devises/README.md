# Bellman-Ford - Arbitrage entre devises

## Lancer le projet

### Compilation
```bash
gcc Arbitrage_devises.c -o arbitrage -lm
```

### Exécution
```bash
./arbitrage
```

## Description

Programme qui détecte les opportunités d'arbitrage entre devises (EUR, USD, JPY) en utilisant l'algorithme de Bellman-Ford pour identifier les cycles négatifs dans un graphe de taux de change.
